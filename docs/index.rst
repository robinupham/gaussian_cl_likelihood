gaussian_cl_likelihood documentation
====================================

Welcome to the documentation for the `gaussian_cl_likelihood <https://github.com/robinupham/gaussian_cl_likelihood>`_ repository, which contains all the code needed to produce the figures in `Upham, Brown & Whittaker 2021, "Sufficiency of a Gaussian power spectrum likelihood for accurate cosmology from upcoming weak lensing surveys", arXiv:2012.06267 <https://arxiv.org/abs/2012.06267>`_.
